# sql-data-analysis
Analyzing business data using SQL and Python to extract insights and answer real-world analytical questions.

## Overview
This project simulates a business dataset and uses SQL queries combined with Python to analyze sales performance, customer behavior, and revenue trends.

## What I Did
- Created a structured dataset and loaded it into a SQL database
- Wrote SQL queries to analyze revenue, customer spending, and product performance
- Used GROUP BY, ORDER BY, and aggregate functions to extract insights
- Integrated SQL with Python using pandas
- Visualized key business metrics using bar charts

## Tools Used
- SQL (SQLite)
- Python
- Pandas
- Matplotlib

## Key Insights
- Revenue varies significantly across regions
- Certain product categories drive higher average spending
- A small group of customers contributes the most revenue
